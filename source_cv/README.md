# Source CV Directory

Place your original CV Word document here.

## Instructions

1. Copy your CV to this directory:
   ```bash
   cp /path/to/your/cv.docx source_cv/cv.docx
   ```

2. The CV should be named `cv.docx` for the default scripts to work

3. If you use a different filename, specify it when running the scripts:
   ```bash
   python scripts/parse_cv.py source_cv/your_filename.docx
   ```

## Note

Your CV files in this directory are ignored by git (see `.gitignore`) to keep your personal information private.
